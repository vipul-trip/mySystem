📋 Step-by-Step Fix Instructions

Step 1: Check Service Status

Open Command Prompt as Administrator and run:

tasklist | findstr -i "mantra"

You should see MantraRDService.exe running

Step 2: Check Port Binding

netstat -ano | findstr :11100

You should see TCP 127.0.0.1:11100 LISTENING

Step 3: Test Direct HTTP Access

curl -v http://127.0.0.1:11100/rd/info

Should return XML response with device information

Step 4: Restart Service (if needed)

taskkill /f /im MantraRDService.exe
Start RD Service as Administrator
